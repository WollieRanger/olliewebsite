2.1.7 Describe which are the appropriate filetypes for websites

These are some if not all the file types that can be used online. They include audio, code, executable files, video and picture.

.AVI | .CSS | .DOC | .EXE | .GIF | .HTML/ .HTM | .JPG/ .JPEG | .MID/ .MIDI | .MP3 | .MPG/ .MPEG | .MOV/ .QT | .PDF | .PNG | .RAM | .RAR | .TIFF | .TXT | .WAV | .ZIP | Streaming Media ]

All have different purposes and uses for websites. The coding file types such as .CSS and .HTML allow content to be displayed on a website. These are obviously particularly important. Picture files such as .JPEG and .PNG allow pictures to be displayed on a website. Sound and video types such as .MID, .MP3, .AVI and .MPEG allow for sound and video to be played on the website. [1]
